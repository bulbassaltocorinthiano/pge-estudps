1Crie sua conta gratuita no Supabase

2Crie as tabelas do banco
No Supabase: SQL Editor → New query → abra o arquivo supabase-schema.sql → cole todo o conteúdo → clique Run. Aguarde "Success. No rows returned".
3Copie suas credenciais
No Supabase: Settings → API → copie a Project URL e a anon public key e cole abaixo.
Verifique a URL e a chave e tente novamente.
